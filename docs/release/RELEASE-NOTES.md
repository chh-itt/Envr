# 发行说明（Release notes）

## 0.1.0（Windows 首发）

**发布日期**：以 Git 标签 / GitHub Release 为准。

### 亮点

- **CLI** `envr`：安装、切换、列出、卸载多语言运行时，JSON 输出与项目级 `.envr.toml` 配置。
- **GUI** `envr-gui`：仪表盘、运行时导航、设置与下载浮层面板（Iced）。
- **Shim** `envr-shim`：核心工具链入口解析（node / npm / python 等），与 CLI 共用配置与运行时根。

### 产物

- `envr-windows-x86_64-0.1.0.zip`：内含 `envr.exe`、`envr-gui.exe`、`envr-shim.exe` 及 `SHA256SUMS.txt`。

### 升级说明

首发版本无旧版迁移说明；后续版本将在此记录破坏性变更与迁移步骤。
